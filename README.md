## 🚀 Projetos em Destaque (Linguagem C)

### 🏪 Sistema "Lojinha" CLI
Um simulador interativo de vendas via terminal que demonstra o controle de fluxo de caixa e estoque:
*   **Gestão Financeira:** Controle de saldo, depósitos e validação de compra por saldo insuficiente.
*   **Mecânica de Estoque:** Geração de estoque aleatório utilizando a função `rand` e decremento automático após vendas bem-sucedidas.
*   **Segurança de Dados:** Prevenção de estoques negativos e carrinhos inconsistentes.

Protótipo para análise de desempenho acadêmico/físico:

## 🧠 Exercícios de Lógica (Lista 100 Algoritmos)

Resoluções selecionadas da lista do Prof. Guanabara, focando em estruturas de dados e modularização:

### 🔹 Passo 08: Vetores (Arrays)
*   **Ex 83:** Preenchimento de vetor com números aleatórios (0-99) e implementação de **algoritmo de ordenação crescente**.
*   **Ex 84:** Manipulação de vetores paralelos para associar nomes e idades, com filtragem de menores de idade.
*   **Ex 85:** Gestão de banco de dados simples para funcionários (Nome, Sexo, Salário) com filtros de gênero e faixa salarial.

### 🔹 Passo 10: Funções (Modularização)
*   **Ex 95 a 99:** Desenvolvimento de bibliotecas de funções matemáticas personalizadas, incluindo `Somador()`, `Media()`, `Maior()` (para 3 números), `SuperSomador()` (soma de intervalos) e `Potencia()`.
*   **Ex 100:** Integração de funções onde o resultado de uma (`Media`) alimenta a lógica da outra (`Situacao`), determinando o status do aluno (Aprovado/Recuperação/Reprovado).
